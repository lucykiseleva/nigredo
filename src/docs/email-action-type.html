Типы действий По своей сути механика Inbox Actions позволяет добавлять в тему письма управляющие кнопки, которые делятся на 4 типа по их способу работы и функционалу: One-click actions — для всего, что делается в один клик. RSVP actions — для мероприятий. Review actions — отзывы для фильмов, заведений, продуктов и сервисов. Go-to actions — для всего, что требует действия при переходе на сайте. Все эти действия работают в стандартном веб-интерфейсе Gmail, а также при просмотре почты через Inbox от Google.
